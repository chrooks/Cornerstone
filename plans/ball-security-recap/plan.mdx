## What shipped

Cornerstone's Ball Security Impact Trait previously approximated turnover economy from three unrelated skills (`passer`, `pnr_ball_handler`, `driver`). This work added a real 22nd skill, **`secure_handler`**, derived from turnover statistics already in the stats blob — Oliver TOV% primary, turnovers-per-touch secondary, `time_of_possession` gates that rise with the tier (4 / 3 / 1.5 / none), and two tier bumps (a skill-curve bump for elite-band TOV% at usage ≥ 28%, and an AST/TO ≥ 2.75 relief valve for playmaking hubs like Jokić whose Oliver TOV% understates them). The trait now reads the skill tier when a profile carries the key, and falls back to the legacy proxy **only** for key-absent profiles (unbackfilled Legends) — a rated "None" never inherits the proxy. Decisions are recorded in ADR 0004 and the throughline's Decision Ledger; the research evidence lives in `docs/research/secure-handler-evidence.md`.

The change is live end to end: Evaluation Version **cohesion-v6** published (22-skill taxonomy, ball_security theoretical max 28.8 → 16.0), thresholds calibrated over three live rounds against a six-player anchor slate (9/9 passing), the review queue triaged (335 flood flags bulk-accepted, 24 bump promotions human-reviewed), and a new Snapshot Release published. A live three-way probe through `POST /api/builder/evaluate` confirmed the fallback contract: key-absent + strong proxy → 7.9, rated-None + strong proxy → 0.0 (no leak), rated-Elite alone → 7.0.

Calibration surfaced and fixed three latent bugs beyond the feature itself: `composite_skill` dropped the evaluator's `review_recommended` (HIGH-confidence skills could never reach the review queue), the stat-leaders resolver only understood `sum` computed stats (three duplicated copies, now one shared module), and distribution warm-up pre-filled default skills before the engine (which would have erased Legend key-absence). Pre-existing dirty files not owned by this work (`.github/`, root `package.json`/`package-lock.json`, `frontend/next.config.mjs`, `frontend/scripts/dev-with-cleanup.mjs`, `RunDiffPreview.tsx`) are excluded from this recap.
